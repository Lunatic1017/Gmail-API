Steps to run the code
1. Install Node module and all the modules used in the project
2. Run the index.js file
3. Run Postman command for google-login 
4. Use the url in the result to verify
5. Verify the user by logging in 
6. Now run the Postman command of user-credentials
7. Running this command will save the tokens in clientcredentials.json 
8. In user-credentials command we also run the url to fetch the email of the user
9. Now we have all the tokens and email we need to send the mail
10.Run the send-mail commad and enter to , subject and message in json format
11.Your email is now sent
